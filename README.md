# SwiftUI Daily Practice

A structured collection of daily SwiftUI practice projects designed to move from absolute basics to advanced professional-grade UI development.

## 🚀 Project Goals
- Master SwiftUI layout and views.
- Deep dive into State Management (`@State`, `@Binding`, `@FocusState`).
- Build real-world components used in top-tier apps.
- Implement premium designs with modern aesthetics.

## 📂 Repository Structure
- `1_Text/`: Deep dive into `Text` and `TextField`. Includes 6 practice projects.
- `images/`: Local assets and design inspirations.

## 🛠 Prerequisites
- **Xcode 15+**
- **iOS 17+** (Recommended for the latest SwiftUI features like `.format` and `axis: .vertical`)

## 📝 How to use this repo
1. Open the `assignments.md` in each folder.
2. Review the **Key Knowledge Areas**.
3. Use the **Visual Inspiration** mockups as your design goal.
4. Try to implement the solution in a new SwiftUI View.
